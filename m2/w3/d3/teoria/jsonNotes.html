Appunti sul formato JSON

JSON: JavaScript Object Notation

JSON è il formato di interscambio dati nelle chiamate moderne HTTP

Lo useremo ogni volta che avremo la necessità di recuperare dati dall'internet o se avremo il bisogno di inviare dei dati a nostra volta al fine di veni memorizzati in un database passando per un server

JSON non è un formato "inventato", è semplicemente la notazione letterale

Qualche rregola speciale: sempre doppi apici e non singoli,le coppie chiave/valore saranno separate da una virgola APPARTE l'ultima, il valori accettabili sono quelli di JS
